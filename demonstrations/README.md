# Usage

1. Install CodeSnap (VSCode extension)

1.1 `npx vsce package`
1.2 `code --install-extension bulk-codesnap-runner-0.0.1.vsix`

or

1.1 `npx vsce package --allow-missing-repository; code --install-extension bulk-codesnap-runner-0.0.1.vsix `

2. Press `Ctrl+Shift+P`, then `Run Bulk CodeSnap`

3. PNG files appear in the output folder
